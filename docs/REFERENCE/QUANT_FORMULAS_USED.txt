1. RANKING FORMULAS (Stock Selection)
================================================================================

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1.1 RANK_GM (Geometric Mean Ranking Metric) - PRIMARY FILTER
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Purpose: Combines 15m momentum and daily trend into single ranking score

Formula:
┌──────────────────────────────────────────────────────────────────────────┐
│                                                                          │
│   g1 = 1 + (pct_vs_15m_sma50 / 100)                                     │
│   g2 = 1 + (pct_vs_daily_sma20 / 100)                                   │
│                                                                          │
│   Rank_GM = ((g1 × g2)^0.5 - 1) × 100                                   │
│                                                                          │
│   Equivalent to: Rank_GM = (√(g1 × g2) - 1) × 100                       │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘

Where:
  • pct_vs_15m_sma50 = ((LTP - SMA50_15m) / SMA50_15m) × 100
  • pct_vs_daily_sma20 = ((LTP - SMA20_daily) / SMA20_daily) × 100

Example:
  If pct_vs_15m_sma50 = +8% and pct_vs_daily_sma20 = +5%:
  g1 = 1.08, g2 = 1.05
  Rank_GM = (√(1.08 × 1.05) - 1) × 100 = (√1.134 - 1) × 100 = 6.48%

TRADING THRESHOLD:
  ╔══════════════════════════════════════════════════════════════════════╗
  ║  MIN_RANK_GM_THRESHOLD = 2.5                                         ║
  ║  Trade Entry: ALLOWED only when Rank_GM > 2.5                        ║
  ║  Stocks with Rank_GM ≤ 2.5 are BLOCKED from entry                    ║
  ╚══════════════════════════════════════════════════════════════════════╝

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1.2 PERCENTAGE VS 15-MINUTE SMA200
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Formula:
┌──────────────────────────────────────────────────────────────────────────┐
│                                                                          │
│   pct_vs_15m_sma50 = ((Last_Price - SMA50_15m) / SMA50_15m) × 100    │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘

Interpretation:
  • Positive value = Price ABOVE 15m SMA200 (bullish momentum)
  • Negative value = Price BELOW 15m SMA200 (bearish momentum)
  • Higher value = Stronger short-term momentum

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1.3 PERCENTAGE VS DAILY SMA50
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Formula:
┌──────────────────────────────────────────────────────────────────────────┐
│                                                                          │
│   pct_vs_daily_sma20 = ((Last_Price - SMA20_daily) / SMA20_daily) × 100 │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘

Interpretation:
  • Positive value = Price ABOVE daily SMA50 (medium-term uptrend)
  • Negative value = Price BELOW daily SMA50 (medium-term downtrend)

2. TECHNICAL INDICATORS
================================================================================

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
2.1 SIMPLE MOVING AVERAGE (SMA)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Formula:
┌──────────────────────────────────────────────────────────────────────────┐
│                                                                          │
│   SMA(n) = (C₁ + C₂ + ... + Cₙ) / n                                     │
│                                                                          │
│   Where: C = Closing price, n = Period                                   │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘

Used SMAs:
  • SMA50_15m  = 50-period SMA on 15-minute candles
  • SMA200_15m = 200-period SMA on 15-minute candles
  • SMA50_daily  = 50-day SMA on daily candles
  • SMA200_daily = 200-day SMA on daily candles

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
2.2 EXPONENTIAL MOVING AVERAGE (EMA)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Formula:
┌──────────────────────────────────────────────────────────────────────────┐
│                                                                          │
│   Multiplier (k) = 2 / (n + 1)                                          │
│                                                                          │
│   EMA_today = (Price_today - EMA_yesterday) × k + EMA_yesterday         │
│                                                                          │
│   Initial EMA = SMA(n) of first n periods                               │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘

Used EMAs:
  • EMA50 (for VCP trend filter)
  • EMA(3), EMA(10), EMA(50) for backtesting strategy

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
2.3 RELATIVE STRENGTH INDEX (RSI)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Formula:
┌──────────────────────────────────────────────────────────────────────────┐
│                                                                          │
│   For each period:                                                       │
│     Gain = max(0, Close_today - Close_yesterday)                        │
│     Loss = max(0, Close_yesterday - Close_today)                        │
│                                                                          │
│   Average Gain = Sum of Gains over n periods / n                        │
│   Average Loss = Sum of Losses over n periods / n                       │
│                                                                          │
│   RS = Average Gain / Average Loss                                       │
│                                                                          │
│   RSI = 100 - (100 / (1 + RS))                                          │
│                                                                          │
│   If Average Loss = 0, then RSI = 100                                   │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘

Standard Period: n = 14

RSI Interpretation:
  • RSI > 70: Overbought territory (potential sell signal)
  • RSI < 30: Oversold territory (potential buy signal)
  • RSI = 50: Neutral momentum

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
2.4 AVERAGE TRUE RANGE (ATR)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Formula:
┌──────────────────────────────────────────────────────────────────────────┐
│                                                                          │
│   True Range (TR) = max(                                                 │
│       High_today - Low_today,                                            │
│       |High_today - Close_yesterday|,                                    │
│       |Low_today - Close_yesterday|                                      │
│   )                                                                      │
│                                                                          │
│   ATR(n) = SMA of TR over n periods                                     │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘

Used for: Volatility measurement in VCP detection

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
2.5 SMA RATIO (Trend Strength)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Formula:
┌──────────────────────────────────────────────────────────────────────────┐
│                                                                          │
│   Ratio_15m_50_200 = SMA50_15m / SMA200_15m                             │
│                                                                          │
│   Ratio_daily_50_200 = SMA50_daily / SMA200_daily                       │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘

Interpretation:
  • Ratio > 1.0: Short-term MA above long-term MA (bullish)
  • Ratio < 1.0: Short-term MA below long-term MA (bearish)
  • Ratio = 1.0: MAs converging (potential trend change)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
3.1 CAPITAL ALLOCATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Configuration:
┌──────────────────────────────────────────────────────────────────────────┐
│                                                                          │
│   TOTAL_STRATEGY_CAPITAL = ₹90,000                                      │
│   CAPITAL_PER_POSITION = ₹3,000                                         │
│   MAX_POSITIONS = 33                                                     │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
3.2 QUANTITY CALCULATION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Formula:
┌──────────────────────────────────────────────────────────────────────────┐
│                                                                          │
│   max_qty_by_capital = floor(CAPITAL_PER_POSITION / LTP)                │
│                                                                          │
│   lots = floor(max_qty_by_capital / lot_size)                           │
│                                                                          │
│   quantity = lots × lot_size                                             │
│                                                                          │
│   Final Quantity = max(0, quantity)                                      │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘

Example:
  CAPITAL_PER_POSITION = ₹3,000
  LTP = ₹150
  Lot Size = 1
  
  max_qty_by_capital = floor(3000 / 150) = 20
  lots = floor(20 / 1) = 20
  quantity = 20 × 1 = 20 shares

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
4.1 POSITION RULES SUMMARY
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

┌────────────┬───────────────┬───────────────┬────────────────────────────────┐
│  Position  │   Stop Loss   │    Target     │        SL Type                 │
├────────────┼───────────────┼───────────────┼────────────────────────────────┤
│     P1     │    -2.5%      │    +5.0%      │    FIXED (from entry)          │
│     P2     │    -2.5%      │    +3.75%     │    TRAILING (from high)        │
│     P3     │    -2.5%      │     None      │    TRAILING (from high)        │
└────────────┴───────────────┴───────────────┴────────────────────────────────┘

P3 = "Runner" position with no target (rides the trend)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
4.2 STOP LOSS FORMULAS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

A) FIXED STOP LOSS (P1):
┌──────────────────────────────────────────────────────────────────────────┐
│                                                                          │
│   Stop_Loss_P1 = Entry_Price × (1 + SL_PCT/100)                         │
│                                                                          │
│   Stop_Loss_P1 = Entry_Price × (1 + (-2.5)/100)                         │
│                = Entry_Price × 0.975                                     │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘

B) TRAILING STOP LOSS (P2 & P3):
┌──────────────────────────────────────────────────────────────────────────┐
│                                                                          │
│   Trail_Stop = Highest_Price_Since_Entry × (1 + SL_PCT/100)             │
│              = Highest_Price_Since_Entry × 0.975                         │
│                                                                          │
│   Final_Stop_Loss = max(Initial_Stop_Loss, Trail_Stop)                  │
│                                                                          │
│   Rule: Stop can only move UP, never DOWN                                │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘

Example:
  Entry Price = ₹100
  Initial SL = ₹100 × 0.975 = ₹97.50
  
  If price rises to ₹110 (highest):
  Trail_Stop = ₹110 × 0.975 = ₹107.25
  Final SL = max(₹97.50, ₹107.25) = ₹107.25

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
4.3 TARGET FORMULAS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

┌──────────────────────────────────────────────────────────────────────────┐
│                                                                          │
│   Target_P1 = Entry_Price × (1 + 5.0/100) = Entry_Price × 1.05          │
│                                                                          │
│   Target_P2 = Entry_Price × (1 + 3.75/100) = Entry_Price × 1.0375       │
│                                                                          │
│   Target_P3 = None (Runner - no target, trails with stop)               │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
4.4 RISK-REWARD RATIOS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

┌────────────┬───────────────┬───────────────┬────────────────────────────────┐
│  Position  │   Risk (SL)   │    Reward     │   Risk:Reward Ratio            │

6. VOLUME ANALYSIS
================================================================================

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
6.1 VOLUME RATIO (5-day vs 200-day)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Formula:
┌──────────────────────────────────────────────────────────────────────────┐
│                                                                          │
│   avg_5_day_volume = Sum of last 5 daily volumes / 5                    │
│                                                                          │
│   avg_200_day_volume = Sum of last 200 daily volumes / 200              │
│                                                                          │
│   Volume_Ratio = avg_5_day_volume / avg_200_day_volume                  │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘

Interpretation:
  • Volume_Ratio > 1.0: Recent volume higher than average (accumulation)
  • Volume_Ratio < 1.0: Recent volume lower than average (distribution)
  • Volume_Ratio > 2.0: Significant volume expansion (breakout potential)

7. GOLDEN CROSS DETECTION
================================================================================

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
7.1 GOLDEN CROSS DEFINITION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

A Golden Cross occurs when:
┌──────────────────────────────────────────────────────────────────────────┐
│                                                                          │
│   Previous Day: SMA50 ≤ SMA200 (Ratio ≤ 1.0)                            │
│   Current Day:  SMA50 > SMA200 (Ratio > 1.0)                            │
│                                                                          │
│   Golden Cross = (Ratio_prev ≤ 1) AND (Ratio_current > 1)               │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
7.2 DAYS SINCE GOLDEN CROSS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Formula:
┌──────────────────────────────────────────────────────────────────────────┐
│                                                                          │
│   Days_Since_GC = Count of trading days from last Golden Cross          │
│                   to current date                                        │
│                                                                          │
│   If no Golden Cross found: Days_Since_GC = NULL                        │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘

Interpretation:
  • Days_Since_GC < 20: Fresh Golden Cross (strong buy signal)
  • Days_Since_GC > 100: Mature trend (may be extended)
  • NULL: No Golden Cross (still in bearish territory)

8. VCP (Volatility Contraction Pattern) - Mark Minervini Style
================================================================================

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
8.1 VCP DETECTION CRITERIA
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Requirements:
┌──────────────────────────────────────────────────────────────────────────┐
│                                                                          │
│   1. TREND FILTER: Price > EMA(50) (uptrend context)                    │
│   2. CONTRACTIONS: Minimum 3 volatility contractions                    │
│   3. HIGHER LOWS: Each successive low must be higher                    │
│   4. ATR CONTRACTION: ATR decreasing over pattern                       │
│   5. BREAKOUT LEVEL: Price approaching resistance                        │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
8.2 VCP STOP-LOSS & TARGET
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Formula:
┌──────────────────────────────────────────────────────────────────────────┐
│                                                                          │
│   VCP_Stop_Loss = Last contraction low (pattern support)                 │
│                                                                          │
│   Risk = Entry_Price - VCP_Stop_Loss                                     │
│                                                                          │
│   VCP_Target = Entry_Price + (3 × Risk)    [3R Target]                  │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
8.3 VCP CONFIDENCE SCORING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Confidence factors:
  • Number of contractions (more = better)
  • ATR contraction ratio
  • Proximity to breakout level
  • Higher lows quality
  • Volume expansion on breakout

================================================================================

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
9.1 GTT ORDER TYPES BY POSITION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

┌────────────┬─────────────────┬────────────────────────────────────────────┐
│  Position  │   GTT Type      │   Structure                                │
├────────────┼─────────────────┼────────────────────────────────────────────┤
│     P1     │   OCO (Two-leg) │   Stop Loss + Target (fixed)              │
│     P2     │   OCO (Two-leg) │   Trailing Stop + Target                  │
│     P3     │   Single-leg    │   Trailing Stop only (no target)          │
└────────────┴─────────────────┴────────────────────────────────────────────┘

GTT_TYPE_OCO = "two-leg" (One-Cancels-Other)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
9.2 GTT TRIGGER PRICES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

OCO GTT Structure:
┌──────────────────────────────────────────────────────────────────────────┐
│                                                                          │
│   Condition 1 (Stop Loss):                                               │
│     Trigger = Stop_Loss_Price                                            │
│     Action = SELL at Market                                              │
│     Activated when: LTP ≤ Stop_Loss_Price                               │
│                                                                          │
│   Condition 2 (Target):                                                  │
│     Trigger = Target_Price                                               │
│     Action = SELL at Market                                              │
│     Activated when: LTP ≥ Target_Price                                  │
│                                                                          │
│   OCO Rule: When one condition triggers, the other is cancelled          │
│                                                                          │
└──────────────────────────────────────────────────────────────────────────┘
